{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} **mimium** (*MInimal-Musical-medIUM*) は音楽の記述/生成に特化したプログラミング言語です。 mimiumは音楽プログラミング言語を音楽科やプログラマーのためのツールとしてだけではなく、ソースコードという形で音楽を配布するための基盤:インフラストラクチャーとなるべく設計され、開発されています。 この言語では、低レベルの信号処理から作曲レベルの長い時間軸を扱う表現まで、サンプル単位での制御処理を含めてシンプルな文法で記述ができます。実行にはLLVMというコンパイラ基盤を採用する事でC++等の低レベル言語に匹敵する処理性能を発揮することが可能です。 これらの仕様や文法はいくつかのモダンな音声処理/音楽向けプログラミング言語、例えば *[Faust](https://faust.grame.fr)* や *[Extempore](https://extemporelang.github.io/)* から影響を受けています。 {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-lightbulb" title="New chair metrics!" %}} The Goldydocs UI now shows chair size metrics by default.
