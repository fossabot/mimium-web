{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} **mimium** (*MInimal-Musical-medIUM*) is a domain specific programming language for describing/generating sound and music. mimium is made to be an infrastructure for distributing music in a form of a source code, not only a tool for musicians and programmers. Its syntax and semantics are technically inspired from several modern programming languages for sound such as *[Faust](https://faust.grame.fr)* and *[Extempore](https://extemporelang.github.io/)*. You can write various expression from very low-level like full-scratch signal processing with a simple syntax and high-performance powered by LLVM. Also you can write note-level processing in the same language with a sample-accurate control. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fab fa-gitter" title="Join Gitter Channel!" url="https://gitter.im/mimium-dev/community" %}} Join mimium Gitter Channel!
